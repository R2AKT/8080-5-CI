# 8080-5-CI
Boot module. For connecting to the debugging board - https://github.com/R2AKT/Stepper.

Allows you to read and write data from/to memory.

Requires Arduino firmware and PC software.
 
Модуль загрузки. Для подключения к плате отладки - https://github.com/R2AKT/Stepper.

Позволяет считывать и записывать данные из/в память.

Необходима прошивка для Arduino и ПО для PC.
